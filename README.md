# UDAPT
UDAPT is a depp learning-based deconvolution model for gene expression deconvolution.

## Pre-conditions
- Python 3.7
- pip 或 conda

## 🛠️ Installation

```bash
# 克隆仓库
git clone https://github.com/ttren-sc/UDAPT.git

# 进入项目目录
cd UDAPT

# 安装依赖
pip install -r requirements.txt

bash'''
